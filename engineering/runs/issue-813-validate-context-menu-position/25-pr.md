# Issue 813 PR: Validate ContextMenu.show Position Coordinates

## Pull Request

https://github.com/Rika-Labs/effect-desktop/pull/832

## Summary

`ContextMenu.show` now validates popup coordinates as finite, non-negative logical pixels before building a host request. The regression tests assert that `NaN`, infinities, and negative coordinates fail as `InvalidArgument` without recording a request, while fractional logical-pixel coordinates still pass through unchanged.
