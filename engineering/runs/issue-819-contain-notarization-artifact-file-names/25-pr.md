Opened PR #941:

https://github.com/Rika-Labs/effect-desktop/pull/941

Summary:

- Treats `artifact.json#fileName` as a basename, not a path authority.
- Rejects traversal, nested path, URL-like, and control-byte file names before notarization tools run.
- Keeps valid app and dmg basename metadata on the existing notarization path.

Local verification before push:

- `bun install --frozen-lockfile`
- `bun run check`
- `bun run typecheck`
- `bun run lint`
- `bun run lint:types`
- `bun prettier --check <touched files>`
- `bun test`
- `cargo fmt --check`
- `cargo clippy --workspace --all-targets -- -D warnings`
- `cargo check --workspace`
- `cargo test --workspace`

Known local-only exception: full `bun run format:check` fails on pre-existing `.devin/config.local.json`, outside the branch diff.
