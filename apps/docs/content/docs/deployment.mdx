---
title: Deployment
description: The documentation site is a normal Next.js 16 application.
---

The docs site lives in `apps/docs` and deploys independently from the desktop playground.

```bash
bun --filter @effect-desktop/docs run build
```

The app uses Fumadocs for content and Next.js App Router for routing. The root page is the public web entrypoint; `/docs` contains the documentation tree.

## Production boundary

Deploy the docs app like a standard Next.js application. The desktop playground remains a separate static renderer artifact embedded by the host.
