---
title: Overview
description: Effect Desktop is a typed framework for building native desktop applications with web renderers.
---

Effect Desktop treats the renderer, runtime, native host, and release tooling as separate systems with explicit contracts.

The framework goal is simple: web UI ergonomics with native capabilities that remain typed, permissioned, observable, and testable.

## What belongs here

- public documentation for the framework;
- deployable web pages for users and contributors;
- examples that explain supported contracts.

## What does not belong here

The desktop playground is not the documentation site. It remains the embedded renderer used by the host smoke path.
