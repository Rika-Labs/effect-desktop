---
title: Getting started
description: Create an Effect Desktop app and run it against the development host.
---

Start with the smallest app that exercises the framework boundary: one renderer, one runtime entrypoint, and one desktop config.

```bash
bun create effect-desktop my-app
cd my-app
bun install
bun run dev
```

The development loop has two moving parts:

- the renderer dev server serves ordinary web UI;
- the native host owns windows, WebViews, permissions, and native services.

## Verify the environment

```bash
bun desktop doctor --config desktop.config.ts
```

Doctor checks the local toolchain and prints typed remediation for missing components.
